    $ cd?

    It's a linux command reference;

    • $ is what you type commands after
    • cd is a command to change directory (in this case, page)
    • and since cd has no arguments (nothing after it), it defaults to the home directory (this page)

    The pages, Projects and Blog, would be the argument to cd, so the command would be cd Projects, which changes the current directory (page) to Projects.

    And now you know the basics of cd!
